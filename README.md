# MIUltra

MIUI12.5 扩展

## 使用条件

- MIUI12.5 稳定版(其他版本自测)
- Android11(其他版本自测)
- Xposed 框架(最优 LSPosed)

## 使用前须知

本模块部分功能开启后可能导致一些不可预估的问题,请了解相关功能后开启.开启则表明你已熟知相关后果且由本模块不可预估的后果需由你自行承担.

## 当前功能：

- [x] 状态栏模糊风格修改(包括模糊半径)
- [x] 非默认主题使用模糊
- [x] 状态栏图标部分强制原生风格
- [x] 桌面第三方组件可拖拽至负一屏
- [x] 桌面抽屉模式顶部删除全部分类
- [x] 桌面抽屉模式分页内删除编辑
- [ ] 更多.....

## 注意事项

安装或更新模块后，请手动重新切换一次模块启用开关，并手动强行停止一次推荐 HOOK 的应用。
